## Medmail

A document search and tagging manager. MacOS desktop version available at [Medmailectron](https://github.com/miboland/medmailectron).

## Running Locally

```bash
$ git clone https://github.com/miboland/medmail.git
$ cd medmail
$ yarn
$ yarn dev
```
